# DroneGroundStation
Ground Station for drones developed with Qt 5.8 on Windows 7

开发环境：

​	Qt 5.8.0 MSVC2013版本，需要安装Visual Studio 2013（及以上版本）

### 如何运行

在Qt Creator中打开项目

在构建设置中，设置构建目录
